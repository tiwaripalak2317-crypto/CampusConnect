# CampusConnect 📚
CampusConnect | HTML CSS JS Web App 

A college management web application built using **HTML, CSS, and  JavaScript**. 

## 🚀 Features
- **DOM Manipulation**: Dynamic tables are generated using JavaScript DOM
- **Local Storage**: All data is saved in browser's Local Storage
- **Data Persistence**: Data remains until manually deleted from Local Storage
- **CRUD Operations**: Add, View, and Delete student records

## 🛠️ Tech Stack
- **Frontend**: HTML5, CSS3
- **Scripting**: JavaScript ES6
- **Storage**: Browser Local Storage
- **DOM**: Document Object Model

## 📂 How to Run
1. Clone this repository
2. Open `index.html` in your browser
3. No backend or installation required

## 📝 Note
This project is purely frontend. Data is stored locally in the browser and will be cleared if browser cache is cleared.
